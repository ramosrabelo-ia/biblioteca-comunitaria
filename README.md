# Biblioteca Comunitária - Checkpoint Python

Projeto acadêmico desenvolvido para a disciplina **Building Data Driven Applications**, do curso de **Data Science** da FIAP.

## Objetivo

Desenvolver um programa em Python que simule o controle de empréstimos de uma biblioteca comunitária.

O sistema permite:

- cadastrar o nome e a idade do usuário;
- exibir uma mensagem de boas-vindas usando f-string e cores;
- selecionar livros por categoria;
- controlar a quantidade de livros emprestados;
- limitar o empréstimo a no máximo 5 livros;
- gerar um relatório final de empréstimos;
- cadastrar a data de entrega de cada livro.
